# Contribuindo para o Control of exaggerations 🚀

Primeiramente, obrigado por se interessar em contribuir! Seguem algumas orientações básicas para manter o projeto organizado.

## Como posso contribuir?

### 1. Relatando Bugs
Se você encontrar um erro, abra uma **Issue** detalhando:
- O que aconteceu.
- Quais passos levaram ao erro.
- Versão do seu Python.

### 2. Sugerindo Melhorias
Ideias para novas funções (como gráficos ou exportação para PDF) são sempre bem-vindas! Abra uma Issue com a tag `enhancement`.

### 3. Enviando Mudanças (Pull Requests)
1. Faça um **Fork** do projeto.
2. Crie uma branch para sua modificação:
   ```bash
   git checkout -b feature/minha-melhoria
